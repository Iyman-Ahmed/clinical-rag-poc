## Category E11 — Type 2 diabetes mellitus
Type 2 diabetes mellitus is reported with category code E11. The fourth, fifth,
and sixth characters identify associated complications. Code as many codes within
category E11 as needed to describe all of the patient's complications.

## Common E11 codes
- **E11.9** — Type 2 diabetes mellitus without complications.
- **E11.22** — Type 2 diabetes mellitus with diabetic chronic kidney disease.
  When E11.22 is reported, also assign a code from category N18 to identify the
  stage of chronic kidney disease (e.g., N18.3 for CKD stage 3).
- **E11.21** — Type 2 diabetes mellitus with diabetic nephropathy.
- **E11.65** — Type 2 diabetes mellitus with hyperglycemia.
- **E11.40** — Type 2 diabetes mellitus with diabetic neuropathy, unspecified.
- **E11.319** — Type 2 diabetes mellitus with unspecified diabetic retinopathy
  without macular edema.

## Sequencing guidance
If the diabetes is associated with a chronic kidney complication, sequence the
diabetes code (E11.22) followed by the appropriate CKD stage code from category
N18. The "with" convention links diabetes to the renal manifestation, so a causal
relationship may be assumed unless documentation indicates the conditions are
unrelated.

## Coding example
A patient with type 2 diabetes and stage 3 diabetic chronic kidney disease is
coded as: E11.22 (Type 2 diabetes with diabetic CKD) + N18.30 (CKD stage 3,
unspecified).
